I chose the Testimonials Grid challenge because I enjoy working with and analyzing grid-based layouts. It provided a good opportunity to practice structuring content using CSS layout techniques and improving my frontend design skills.

Approach:

II began by creating the overall layout structure using a main container. After that, I divided the layout into left and right sections using Flexbox.

Next, I focused on structuring the content:

a) The right column was divided into multiple rows to match the design structure.

b) The left column remained as a single section.

c) According to the target layout, there are two main rows, and each row is further divided into two columns (left and right).

Once the layout structure was ready, I added the testimonial images, headings, and descriptive text. I then applied styling such as spacing, alignment, and formatting to make the layout closer to the target design.

Challenges.

One of the main challenges was maintaining consistent spacing and alignment across the layout. I initially struggled with correctly applying Flexbox properties and adjusting padding and margins to ensure uniform formatting.

Another difficulty was setting the width and margin of the left column so that it aligned properly with the rest of the grid and matched the target layout.

I addressed these challenges by debugging the layout using browser developer tools, experimenting with different spacing values, and refining the Flexbox structure to achieve a closer match to the original design. I am still continuing to refine the layout to make it even more accurate.
